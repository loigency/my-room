# 🎨 房间素材说明

将你的 PNG 素材放入此文件夹，刷新浏览器即可自动替换程序化 placeholder。

## 素材清单

| 文件名 | 对应物品 | 建议尺寸 (W×H) | 说明 |
|--------|---------|---------------|------|
| `background.png` | 整个房间背景 | 1600×900 | 可选，放这里会自动替换墙壁+地板 |
| `window.png` | 窗户 | ~360×440 | 右侧的拱形窗 |
| `painting-1.png` | 墙上画框 1 | ~180×230 | 左侧墙上的画框 |
| `painting-2.png` | 墙上画框 2 | ~160×200 | 左侧墙上的画框 |
| `bookshelf.png` | 书架 | ~240×250 | 左侧墙边的书架 |
| `easel.png` | 画架 | ~140×200 | 画架+画布 |
| `desk.png` | 书桌 | ~380×80 | 中央书桌 |
| `chair.png` | 椅子 | ~60×100 | 书桌旁的椅子 |
| `lamp.png` | 台灯 | ~80×80 | 桌上的台灯（含光晕） |
| `typewriter.png` | 打字机 | ~100×50 | 桌上的打字机 |
| `desk-photos.png` | 桌上照片 | ~80×60 | 桌上的相框 |
| `desk-books.png` | 书本 | ~80×40 | 桌上叠放的书 |
| `rug.png` | 地毯 | ~700×180 | 地板上的地毯 |
| `plant.png` | 绿植 | ~80×120 | 角落的盆栽 |
| `clock.png` | 挂钟 | ~80×80 | 墙上的钟 |

## 素材要求

- **格式**：PNG（支持透明通道）
- **风格**：统一色调，暖色系、油画/厚涂质感、有纹理
- **光影**：注意光源方向统一（窗户在右侧，主光从右上打来）
- **边缘**：素材边缘建议做半透明羽化，与背景融合更自然

## 你不需要一次做完所有素材

系统会自动用程序化图形填充还没有素材的物品。你可以一个接一个地添加，随时刷新看效果。

## 如何获取素材

- 用 ChatGPT / Gemini 生图后抠图
- 在 Unsplash、Pexels 找免版权素材后裁切调色
- 自己拼贴合成（你擅长的）
- 扫描/拍摄实物后抠图处理
